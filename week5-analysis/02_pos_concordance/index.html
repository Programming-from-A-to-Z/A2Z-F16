Parts of Speech Concordance

This page demonstrates a "parts of speech" concordance by counting the number of occurrences of all parts of speech in a body of text. Drag and drop any files to see a list of all POS sorted by counts. File issues and view source on github.

This is built on top of the RiTaJS library by Daniel Howe. What the tags mean can be found here.
